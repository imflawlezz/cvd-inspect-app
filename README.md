# CVD Inspect

CVD Inspect is an application for assessing cardiovascular disease (CVD) risk based on lipid profile results.

## Live Demo

[https://cvd-inspect-app.vercel.app/](https://cvd-inspect-app.vercel.app/)

## About the Project

The application allows users to input their medical data and receive a cardiovascular risk assessment. It was built using modern web and mobile technologies.

## Technologies

- **Frontend:** React, Ionic Framework
- **Build Tool:** Vite
- **Mobile Runtime:** Capacitor (iOS and Android support)
- **Language:** TypeScript
